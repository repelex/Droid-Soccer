\subsection{Detailed Description}
Container for user authentication in \hyperlink{namespace_photon}{Photon}. 

Set Auth\+Values before you connect -\/ all else is handled. 

On \hyperlink{namespace_photon}{Photon}, user authentication is optional but can be useful in many cases. If you want to Find\+Friends, a unique ID per user is very practical.

There are basically three options for user authentification\+: None at all, the client sets some User\+Id or you can use some account web-\/service to authenticate a user (and set the User\+Id server-\/side).

Custom Authentication lets you verify end-\/users by some kind of login or token. It sends those values to \hyperlink{namespace_photon}{Photon} which will verify them before granting access or disconnecting the client.

The \hyperlink{namespace_photon}{Photon} Cloud Dashboard will let you enable this feature and set important server values for it. \href{https://www.exitgames.com/dashboard}{\tt https\+://www.\+exitgames.\+com/dashboard}
