Requires a \hyperlink{class_photon_view}{Photon\+View}. Disables itself on Game\+Objects that are not owned on Start.

Speed affects movement-\/speed. Jump\+Force defines how high the object \char`\"{}jumps\char`\"{}. Jump\+Timeout defines after how many seconds you can jump again.
