\subsection{Detailed Description}
Basic G\+UI to show traffic and health statistics of the connection to \hyperlink{namespace_photon}{Photon}, toggled by shift+tab. 

The shown health values can help identify problems with connection losses or performance. Example\+: If the time delta between two consecutive Send\+Outgoing\+Commands calls is a second or more, chances rise for a disconnect being caused by this (because acknowledgements to the server need to be sent in due time).
